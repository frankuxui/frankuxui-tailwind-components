#### With Rayphe prism

Muestra un diálogo de alerta con un mensaje opcional, y aguardará hasta que el usuario cierre la ventana de diálogo.

```jsx
<button className='flex h-10 text-base px-6 flex-none items-center justify-center flex-row font-normal rounded-md no-underline transition-colors bg-blue-600 text-white hover:bg-blue-700 dark:bg-red-500'>Button</button>
```

#### Basic buttons

```jsx
<button className='flex h-10 text-base px-6 flex-none items-center justify-center flex-row font-roboto font-normal rounded-md no-underline transition-colors bg-slate-100 text-slate-700 hover:bg-slate-200 hover:text-slate-900'>Button</button>
```

#### Filled buttons

Muestra un diálogo de alerta con un mensaje opcional, y aguardará hasta que el usuario cierre la ventana de diálogo.

```jsx

```
